

Message Window
	To Send:
		"AllMessagesRead" : All messages on this board have been read
			Sets state on "MailBox" to "NewMessages";
		"NewMessages" : There are new messages.
			Sets state on "MailBox" to "default";

	Have a text property called "Name" which is the name of the fleet
